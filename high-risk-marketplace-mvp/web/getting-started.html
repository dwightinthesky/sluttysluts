後端快速開始

三步驟啟動後端並跑完整 smoke test

啟動步驟

  1. 進入 backend 目錄並安裝套件
  2. 啟動 API 服務（預設 8080）
  3. 在另一個 terminal 執行 smoke test
cd ../backend
npm install
npm start

# another terminal
npm run smoke

注意事項

  • 所有受保護 API 需帶 x-user-id header。
  • 預設內建 demo 帳號：buyer-demo / seller-demo / admin-demo / ops-demo。
  • 此版本為 MVP 模擬流程，支付與 KYC 為測試 stub。
